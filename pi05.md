## Installation
Currently only usable with everse uv environment. Inside everse uv environment, 
```
cd external/openpi
uv pip install -e .
```
Replace the default transformers module inside your environment with the custom version:
```
cp -r ./src/openpi/models_pytorch/transformers_replace/* <path_to_your_uv_env>/lib/python3.11/site-packages/transformers/
```

## Getting pretrained checkpoint
From EgoVerse directory
```
mkdir egomimic/algo/pi_checkpoints
uv pip install gsutil
gsutil -m cp -r gs://openpi-assets/checkpoints/pi05_base egomimic/algo/pi_checkpoints/
```

Convert JAX checkpoint into Pytorch
```
cd external/openpi
uv run --active examples/convert_jax_model_to_pytorch.py \
  --config_name pi05_aloha \
  --checkpoint_dir ../../egomimic/algo/pi_checkpoints/pi05_base \
  --output_path ../../egomimic/algo/pi_checkpoints/pi05_base_pytorch
```
